Bootstrap is a popular open-source front-end framework that is used for designing and developing responsive and user-friendly websites and web applications. The primary goal of Bootstrap is to simplify and streamline the process of building consistent and visually appealing web interfaces.

Sass, which stands for "Syntactically Awesome Style Sheets," is a preprocessor scripting language that is used to enhance the capabilities of regular CSS (Cascading Style Sheets). It provides a more efficient and powerful way to write and manage styles for web development projects. The primary goal of Sass is to make CSS coding more maintainable, modular, and easier to work with, especially for larger and more complex projects.

  • Familiarity in utilizing Sass variables and mixins to enhance code reusability and maintainability.
  • Basic knowledge in optimizing code readability by employing proper nesting techniques for nested CSS rules.
  • Familiar at optimizing stylesheets for performance and reducing redundant code using Sass features.

  • Familiarity with embedding JavaScript expressions in JSX.
  • Experience with CSS-in-JS libraries (e.g., styled-components, emotion) or traditional CSS for styling components.
  • Familiarity with useState hook for managing state in functional components.
  • Familiarity in using React DevTools for debugging and inspecting component hierarchy.
  • Familiarity with deploying React applications to hosting platforms like Netlify, Vercel, or GitHub Pages.
